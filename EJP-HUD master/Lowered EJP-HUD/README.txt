Thanks for installing EJP-HUD.

Version: 1
Released: 25 December 2014

Notes:

-Be sure to install every font in the fonts folder before using this HUD.

-To apply any of the customizations, copy the desired file and paste it in EJP-HUD/resource/ui. 
!This is EJP-HUD/scripts for the hudlayout.res file. 

-To enable any of the provided crosshairs, go to hudlayout.res and change the visible value to '1' of the section of the correct crosshair.
The name of the crosshair usually describes the crosshair, so try starting from there.
